VS Code 常用快捷键大全（以 Windows/Linux 为主，macOS 对应键已注明）

1. 通用操作

功能 Windows/Linux macOS
打开命令面板 Ctrl + Shift + P / F1 Cmd + Shift + P
快速打开文件 Ctrl + P Cmd + P
打开设置 Ctrl + , Cmd + ,
打开键盘快捷键设置 Ctrl + K Ctrl + S Cmd + K Cmd + S
新建窗口 Ctrl + Shift + N Cmd + Shift + N
关闭窗口 Ctrl + Shift + W Cmd + Shift + W
保存文件 Ctrl + S Cmd + S
全部保存 Ctrl + K S Cmd + Option + S

2. 文件与编辑器管理

功能 Windows/Linux macOS
新建文件 Ctrl + N Cmd + N
关闭当前文件 Ctrl + W Cmd + W
关闭所有文件 Ctrl + K Ctrl + W Cmd + K Cmd + W
重新打开已关闭文件 Ctrl + Shift + T Cmd + Shift + T
切换编辑器组 Ctrl + 1/2/3 Cmd + 1/2/3
拆分编辑器 Ctrl + \ Cmd + \
切换侧边栏 Ctrl + B Cmd + B
全屏切换 F11 Ctrl + Cmd + F

3. 基础编辑

功能 Windows/Linux macOS
剪切行（无选区时） Ctrl + X Cmd + X
复制行（无选区时） Ctrl + C Cmd + C
删除行 Ctrl + Shift + K Cmd + Shift + K
向上/向下移动行 Alt + ↑ / ↓ Option + ↑ / ↓
向上/向下复制行 Shift + Alt + ↑ / ↓ Shift + Option + ↑ / ↓
在下方插入空行 Ctrl + Enter Cmd + Enter
在上方插入空行 Ctrl + Shift + Enter Cmd + Shift + Enter
撤销 Ctrl + Z Cmd + Z
重做 Ctrl + Y Cmd + Y
注释/取消注释行 Ctrl + / Cmd + /
多行注释 Shift + Alt + A Shift + Option + A
格式化文档 Shift + Alt + F Shift + Option + F

4. 搜索与替换

功能 Windows/Linux macOS
查找 Ctrl + F Cmd + F
替换 Ctrl + H Cmd + Option + F
全局搜索 Ctrl + Shift + F Cmd + Shift + F
选中当前词的所有匹配 Ctrl + D Cmd + D
选中所有匹配项 Ctrl + Shift + L Cmd + Shift + L

5. 导航与跳转

功能 Windows/Linux macOS
转到行号 Ctrl + G Cmd + G
转到定义 F12 F12
转到符号 Ctrl + Shift + O Cmd + Shift + O
显示问题面板 Ctrl + Shift + M Cmd + Shift + M
转到上一个/下一个错误 F8 / Shift + F8 F8 / Shift + F8

6. 多光标与选择

功能 Windows/Linux macOS
插入光标（点击） Alt + 点击 Option + 点击
在上方/下方插入光标 Ctrl + Alt + ↑ / ↓ Option + Cmd + ↑ / ↓
列选择（框选） Shift + Alt + 拖动 Shift + Option + 拖动

7. 调试

功能 快捷键
开始/继续调试 F5
停止调试 Shift + F5
步过 F10
步入 F11
步出 Shift + F11
切换断点 F9

8. 终端

功能 Windows/Linux macOS
显示/隐藏终端 `Ctrl + `` `Ctrl + ``
新建终端 `Ctrl + Shift + `` `Cmd + Shift + ``
