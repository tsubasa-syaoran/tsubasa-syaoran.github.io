原文：小白研究所 VPS-1Panel 教程

一、什么是 1Panel 面板？

1Panel 是一款现代化、开源免费的 Linux 服务器运维管理面板。 (xbiit.net)

它可以：

  • 通过网页图形界面管理 VPS
  • 不再依赖复杂 Linux 命令
  • 基于 Docker 容器技术
  • 应用彼此隔离
  • 更安全、更轻量

官方：

  • 1Panel 官网
  • 1Panel GitHub
